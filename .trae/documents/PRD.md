## 1. 产品概述
多人协作搬家规划与智能物品追踪平台，面向家庭搬家场景提供全流程数字化解决方案。通过AI识别、三维装箱算法、路线优化等技术，解决搬家过程中协作混乱、物品丢失、空间浪费、费用不透明等痛点。

- 核心价值：多人协同打包效率提升60%，空间利用率提升35%，物品追踪准确率99%+
- 目标用户：需要搬家的家庭用户、小型搬家公司、搬家团队

## 2. 核心功能

### 2.1 用户角色
| 角色 | 加入方式 | 核心权限 |
|------|----------|----------|
| 项目负责人 | 创建项目 | 项目全权限管理、成员邀请/移除、费用审批 |
| 协作成员 | 邀请链接/项目码加入 | 物品录入、打包操作、任务执行、状态更新 |
| 查看者 | 只读邀请 | 查看进度、看板、费用预估，无编辑权限 |

### 2.2 功能模块
1. **项目总览看板**：甘特图展示各房间打包进度、任务分配、异常标记、全局统计
2. **项目与成员管理**：创建/编辑项目、成员邀请、角色权限分配、项目码生成
3. **纸箱与物品管理**：二维码生成与扫描、物品录入、拍照识别、属性编辑、房间分组
4. **AI智能识别**：TensorFlow.js物品类别识别、自动预填信息、置信度展示、手动修正
5. **三维装箱规划**：纸箱/物品尺寸录入、货车容量配置、空间计算、装载顺序建议、3D可视化
6. **路线规划**：新旧地址配置、多站点优化、搬运顺序、交通因素、按房间装载优先级
7. **异常监控**：超重警报、易碎品未固定标记、遗漏物品检测、进度异常提醒
8. **费用预估**：货车容量匹配、基础费用计算、保险建议、人工成本、对比报价

## 3. 核心流程

用户创建搬家项目 → 配置基本信息与货车规格 → 邀请协作成员分配角色 → 生成纸箱二维码打印贴箱 → 扫码纸箱+拍照物品 → AI识别物品类别预填信息 → 手动确认/修正物品属性 → 三维装箱算法计算装载方案 → 生成按房间分组的装载优先级 → 规划新旧地址最优搬运路线 → 甘特图实时更新各房间打包进度 → 异常监控(超重/易碎品)警报 → 生成费用预估与保险建议 → 搬家完成后导出清单

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深青蓝(#0F766E)作为主色，代表专业与信任；琥珀橙(#F59E0B)作为强调色，用于警示和重要操作
- **辅助色**：翡翠绿(#10B981)表示完成/正常；玫红(#F43F5E)表示异常/超重；紫罗兰(#8B5CF6)表示AI识别
- **按钮风格**：圆角12px，hover时有微妙上浮+阴影加深，active有轻微凹陷感
- **字体方案**：标题使用Space Grotesk(现代几何无衬线)，正文使用Noto Sans SC(中文可读性)
- **布局风格**：卡片式布局+左侧导航，卡片使用柔和阴影+16px圆角，信息密度中等
- **图标风格**：Lucide图标库线性风格，关键操作使用填充版本区分

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 总览看板 | 甘特图区域 | 彩色时间条、渐变进度、房间色块分组、悬停详情浮层 |
| 总览看板 | 统计卡片 | 玻璃拟态背景、数字动效、趋势小箭头、图标角标 |
| AI识别面板 | 识别结果 | 置信度渐变进度条、类别标签动效、脉冲扫描线动画 |
| 装箱规划 | 3D视图 | 等轴测投影、半透明纸箱块、鼠标悬停高亮、旋转交互 |
| 异常告警 | 告警条目 | 左侧色条标识级别、闪烁微动效、快速处理按钮 |
| 费用预估 | 费用卡片 | 展开折叠动画、费用项逐项出现、对比表格斑马纹 |

### 4.3 响应式
- Desktop-first设计，主断点1024px(平板)和640px(手机)
- 桌面：左侧固定导航+主内容区双栏布局
- 平板：导航折叠为顶部汉堡菜单，甘特图支持横向滚动
- 手机：卡片纵向堆叠，扫码/拍照全屏模式，底部Tab导航
- 触摸优化：按钮最小48x48px触控区，滑动手势支持甘特图调整
